
### Install Dependencies

**Frontend:**
```bash

```

**Backend:**
```bash
cd ../server
npm install
npm start
```

### Setup Database

**Start MongoDB:**
```bash
# Windows
"C:\Program Files\MongoDB\Server\8.2\bin\mongod.exe" --dbpath "C:\data\db"

# macOS

# Linux
mongod --dbpath /var/lib/mongodb
```
