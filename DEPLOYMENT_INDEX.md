### Deployment Options
1. **Render.com** (RECOMMENDED)
   - Free tier available
   - Both frontend and backend
   - Auto-deploy from GitHub
   - 5 minutes setup

2. **Vercel + Render**
   - Vercel for frontend (best performance)
   - Render for backend
   - Bit more setup

3. **Heroku**
   - Full-stack deployment
   - Requires credit card
   - More configuration
